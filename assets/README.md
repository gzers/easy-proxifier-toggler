# 图标资源说明

本目录包含 Proxifier Toggler 使用的所有图标资源。

## 文件说明

### 托盘图标（自动状态切换）
- `icon.png` / `icon.ico` - 活动状态图标（蓝色，Proxifier 运行中）
- `icon_inactive.png` / `icon_inactive.ico` - 非活动状态图标（灰色，Proxifier 已停止）

### 其他资源
- `icon.png` / `icon.ico` - 通用图标（用于窗口标题栏等）
- `gzgg-logo.gif` - 其他资源文件

## 图标特性

- **圆形设计**: 现代化的圆形图标，符合 Windows 11 设计语言
- **透明背景**: 所有图标均使用透明背景，适配各种主题
- **多尺寸支持**: ICO 文件包含多个尺寸（16x16 到 256x256），确保在不同 DPI 下清晰显示
- **状态指示**: 集成了开关元素设计，直观表达切换功能

## 自定义图标

如果你有自己设计的图标，只需：
1. 将图标命名为 `icon.png` 和 `icon_inactive.png`
2. 放置在本目录下
3. 确保图标为透明背景的 PNG 格式
4. 推荐尺寸：256x256 像素

程序会自动加载这些图标文件。

## 技术细节

- **格式**: PNG（带 Alpha 通道）和 ICO（多尺寸）
- **推荐尺寸**: 256x256 像素（会自动缩放）
- **颜色模式**: RGBA
- **依赖库**: PIL/Pillow
